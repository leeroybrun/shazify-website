Magic!

Shazify auto-magically search all your Shazam tags on Spotify and add them to a new playlist.

Manual search

In case we don't find the right song on Spotify, you can manually search and select the right match directly in Shazify.

Preview songs

You can preview songs directly in Shazify to be sure it's the one you Shazamed in the first place.

Best of all, Shazify is totally free and Open Source! Feel free to check the source code on Github.
